  Factorial Function.

Write a function that accepts parameter n and returns factorial of n.
Code:-
function factorial(a){
    if(a !== 0){
    	let ans = 1;
    	for(let i=2; i<=a; i++){
    		ans = ans * i;
    	}
    	return ans;
    }
};
						
console.log(factorial(a));

  2.

You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps).
Code:-
let s="abcadeecfb"
let map = new Map()
for(let i=0;i < s.length;i++){
    if(map.has(s[i])){
        let value = map.get(s[i])
        map.set(s[i], value+1)
    }
    else{
        map.set(s[i], 1)
    }
}
for(let [k,v] of map){
    console.log(k+"="+v);
}
